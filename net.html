Gl*tch in the Ro*tine



Programs: Atom, Github

Medium: HTML, CSS, ThreeJS

Dimensions: webpage fullscreen


Gl*tch in the Ro*tine was a mock online exhibit that my friend and I created for the video installation project in Art 75. The objective for this assignment was to create and form an exhibit by planning out out the layout of the display as well as incorporating a video as your main attraction. For us, we wanted to try to do a VR video and create an online exhibit to display this video.




Eye Site



Programs: Atom, Github

Medium: HTML, CSS

Dimensions: webpage fullscreen


This website was made during Art 75 and features a few of the projects that I worked on in the Gl*tch in the Ro*tine online exhibit. The overall theme is somewhat like a blog similar to those on Tumblr, and was purposely made to look "safe" by incorprating a cute and simple design. This is meant to distract visitors from the underlying dangers some websites can cause, like recording your data or infecting your laptop with viruses. This website was also the prototype for my portfolio.




Marching Along



Programs: Atom, Github

Medium: Javascript

Dimensions: 810 x 810

This was first time experimenting with javascript that I learned during Art 74. The design was intentially made to look very cute and relaxing - a stark contrast to most of the themes I usually present in my projects. Before I knew it, I found myself enjoying the process and even had fun making it.The latest edition to my Art 74 ePortfolio is my 5th project, the ArtWare. I wanted this project to match the overall aesthetic and theme of my website, so I chose matching colors and cute images.

For this project, I experimented with javascript and played around with the code mostly from the book Getting Started With p5.Js. This was alot harder to learn compared to HTML and css because it is a completely different language than the two. I did have alot of trouble adding a few more elements to my project but in the end I chose to keep it sweet and simple.
